# Presentación del Proyecto — Quantum Guide

## Para qué existe
Quantum Guide existe para atender consultas de potenciales clientes de Quantum Motors en canales digitales con un estilo **casi humano y amigable**, y con información confiable sobre:
- vehículos en stock (por región)
- ahorro económico
- no emisiones de CO2
- sucursales
- disponibilidad de Test Drive
- contenido multimedia (fotos y videos)

## Dónde vive la mejor experiencia
La mejor experiencia se entrega en la **Web Showroom** (demo):
- Chat + panel visual reactivo.
- Interacción por categorías y filtros.
- Cierre de Test Drive.

Los canales sociales (WhatsApp/Instagram/TikTok) se consideran extensión natural para fase posterior, usando el mismo motor y las mismas fuentes de verdad.

## Qué entrega la demo
- Demostración de asesoría guiada.
- Recomendaciones basadas en datos.
- Visualización de modelos y resumen técnico.
- Registro de Test Drive para operación.
- Memoria de chat (opcional con Supabase).
